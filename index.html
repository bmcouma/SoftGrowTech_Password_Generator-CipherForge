    Strength Guide

    • Very Weak Under 28 bits — trivially crackable
    • Weak 28 – 35 bits — vulnerable to brute force
    • Fair 36 – 59 bits — moderate protection
    • Strong 60 – 95 bits — good for most accounts
    • Very Strong 96+ bits — enterprise / high-security

    Security Tips

    • Use a unique password for every account.
    • Aim for 96+ bits of entropy for critical accounts.
    • Store passwords in a reputable password manager.
    • Enable two-factor authentication wherever available.
    • Never reuse a password from a known data breach.
